React Router

官方的話: React Router 是完整的React 路由解決方案，React Router 保持UI 與URL 同步。它擁有簡單的API 與強大的功能例如代碼緩衝加載、動態路由匹配、以及建立正確的位置過渡處理。你第一個念頭想到的應該是URL，而不是事後再想起。
